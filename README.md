# LearningVue
Learning Vue
先确定有安装过nood.js Vue webpack

vue init webpack my-project
最新版的都会自动装 路由vue-router，如果没有自动装的话npm install vue-router
my-project可以自己命名

cd my-project
npm install

然后可以把我这里的src文件夹覆盖原来的src

npm run dev
可以自己调试测试，没问题就套用到自己的项目里。
